AutoSkill

SkillSets provided with permissions/commands

  By default, the configuration generated will max all skills when the player is connecting or died

Commands

  • /skills show the command usage and skill list relative to your permissions
  • /skills <SkillSetName> set your skills according to the configuration
 

Permissions

  • skills for /skills
  • skills.<SkillSetName> for /skills <SkillSetName>
 

AutoSkillConfiguration

Xml tag name Default value Description
StorageType file file: to save on the disk memory: to save into the memory (when the server is shutdown, nothing is saved)
FilePath EnabledAutoSkill.xml The path where the file is located (only StorageType: file)
PeriodicSave 10000 The delay between saves in milliseconds (only StorageType: file (for now))
 

SkillSet

Xml tag name Example value Description
Name offense The name of the skillset
Default true Define which SkillSet will be automatically assigned to player (you can set all your SkillSet/Default to false)
Skills 
<Skill Name="Agriculture" Level="7" />
          <Skill Name="Engineer" Level="3" />
The skill list to be updated. All skills not referenced in this list will be Leveled to 0 by default.
